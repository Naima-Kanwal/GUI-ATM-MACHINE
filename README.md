ATM Machine GUI Project (C++ / Qt)

Description:
This is a GUI-based ATM Machine simulation developed using C++ and Qt Creator.
The project allows a user to log in using a PIN and perform basic ATM operations through a graphical interface.

Features:
PIN-based login system
Check account balance
Deposit money
Withdraw money
Logout functionality
User-friendly GUI using Qt Widgets

Technologies Used:
Language: C++
Framework: Qt (Qt Widgets)
IDE: Qt Creator

Concepts Used:
Class and Object
Constructor and Destructor
Encapsulation
Inheritance
Event-driven programming
Signals and Slots (Qt)
GUI programming

Project Structure:
ATM_GUI_Project/
│
├── main.cpp
├── mainwindow.h
├── mainwindow.cpp
├── account.h
├── account.cpp
├── mainwindow.ui
├── ATM_LOGIN.pro
└── README.md

How to Run:
Open the project in Qt Creator
Build the project
Run the application
Enter the PIN to log in and use ATM features

Default Login Details:
PIN: 1234
Initial Balance: 5000

Educational Purpose:
This project is created for learning purposes to understand C++ Object-Oriented Programming and GUI development using Qt.
It was developed as a semester-end project for Programming Fundamentals Lab.
